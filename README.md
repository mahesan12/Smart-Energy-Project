# Smart Energy Monitoring Project

## Project Overview

This project is a smart energy monitoring system designed to measure and analyse the energy use of a small electrical device, 
such as a fan or motor. The system uses electronic sensors, Arduino data collection, and Python analysis to calculate voltage, 
current, power, and efficiency. The aim is to investigate how energy is used and identify ways to reduce unnecessary power consumption.

## Project Aim

The aim of this project is to design, test, and analyse a smart energy system 
that can monitor electrical performance and support energy-saving decisions.

## Key Features

- Measures voltage and current from a test circuit
- Uses Arduino to collect sensor data
- Stores test readings in CSV format
- Uses Python to analyse power and efficiency
- Supports comparison between different operating conditions
- Provides evidence for sustainable energy management and project testing
